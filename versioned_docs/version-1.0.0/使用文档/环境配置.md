# 环境配置

wxauto相关环境配置

## 一、环境要求

- 系统：Windows 10\|11\|Server2016+
- Python：3.7+（不支持3.7.6和3.8.1）
  
:::warning 注意
**python版本不支持3.7.6和3.8.1**
:::

## 二、获取wxauto

控制台运行以下命令安装wxauto
```shell
pip install wxauto
```

## 三、微信版本

本项目目前默认分支为**微信3.9.8.15**版本，使用前请先检查自己电脑微信是否为该版本，版本不同可能由于UI问题导致某些功能无法正常调用

:::tip 注：
如果你的微信版本可以用的话，也不需要过多纠结这个
:::
